# Desafio técnico - Easysecrets

Este repositório contém a resolução do desafio técnico proposto pela Easysecrets.

### Sobre o Projeto
Aplicativo React com TypeScript que transforma dados de vendas em visualizações gráficas interativas, com modo dark/light e filtros dinâmicos.


## 🛠️ Tecnologias Utilizadas

**React**	Biblioteca JavaScript para construção de interfaces
**TypeScript**	Adiciona tipagem estática ao JavaScript
**Vite**	Ferramenta de build rápida para frontend moderno
**Chart.js + react-chartjs-2**	Biblioteca para visualização de dados gráficos
**CSS Modules**	Para estilização componentizada
**react-icons**	Ícones SVG populares como componentes React
**npm**	Gerenciador de pacotes do Node.js

## 📦 Como Rodar o Projeto

Clone o repositório:

**bash**
git clone https://github.com/seu-usuario/desafio-tecnico-easysecrets.git


Acesse a pasta do projeto:
**bash**
cd desafio-tecnico-easysecrets
Instale as dependências:

**bash**
npm install
Execução
Para iniciar o servidor de desenvolvimento:

**bash**
npm run dev

### Pré-requisitos

Certifique-se de ter as seguintes ferramentas instaladas:

* Node.js (versão LTS recomendada, ex: v18.x ou v20.x)
* npm (gerenciador de pacotes do Node.js) ou Yarn

### Instalação

git clone https://github.com/seu-usuario/desafio-tecnico-easysecrets.git
cd desafio-tecnico-easysecrets

Para iniciar o servidor de desenvolvimento:

npm run dev

http://localhost:5173

### Funcionalidades Implementadas

Principais Características
*Visualização de dados com gráficos interativos (barras e linhas)
*Modo dark/light com toggle acessível
*Filtros dinâmicos por tipo de produto
*Responsividade para diferentes tamanhos de tela
*Feedback visual para ações do usuário

Diferenciais
*Tipagem estática com TypeScript
*Componentização reutilizável
*Estrutura de pastas organizada
*Documentação completa
